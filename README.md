# Hospital Management System

## Project Overview

The Hospital Management System is a database management project developed to manage and organize hospital-related information efficiently. The system stores and manages data related to departments, doctors, patients, appointments, admissions, pharmacy, prescriptions, billing, laboratory tests, rooms, staff, and medical records.

The database is designed using MySQL and includes primary keys, foreign keys, unique constraints, NOT NULL constraints, DEFAULT values, CHECK constraints, and AUTO_INCREMENT fields to maintain data integrity and consistency.

## Features

- Manage hospital departments
- Manage doctor information
- Store patient records
- Manage appointments
- Manage patient admissions
- Manage pharmacy and medicines
- Manage prescriptions
- Manage billing information
- Manage laboratory tests
- Manage hospital rooms
- Manage staff records
- Maintain medical records
- Use SQL queries for data retrieval and management
